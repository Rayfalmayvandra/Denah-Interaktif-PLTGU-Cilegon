# Folder Foto untuk b40 (Laboratory)

Letakkan foto-foto untuk bangunan **Laboratory (b40)** di dalam folder ini.
Format foto yang didukung: `.jpg`, `.jpeg`, `.png`, `.webp`

**Saran penamaan file:**
- `thumb.jpg` atau `thumb.png` (akan otomatis dijadikan foto utama / thumbnail)
- Foto tambahan lainnya bebas (misal `lab-1.jpg`, `foto2.jpg`, dll) akan otomatis masuk ke dalam galeri foto bangunan ini saat diklik.
